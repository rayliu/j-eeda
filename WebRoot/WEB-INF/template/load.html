平安易贷

“平安易贷”是中国平安集团推出一个新型贷款产品，是针对急用钱→医疗---生意周转---应急现金--结婚--装修---购物等开展无抵押，无担保贷款业务的信用保证保险业务；现与平安银行、中国光大银行、农业银行合作帮助广大居民快速获取银行无抵押贷款。

贷款特点：

审批快------1—2天放款

门槛低------无需担保，无需抵押

期限长------分期还款，期限1个月—36个月

额度高------按借款人收入核定，最高可达收入12倍

贷款产品如下：

★ 薪金贷款 ——上班族最方便的贷款渠道（注：2—15万）

申请条件：1、中国人民银行有个人征信记录显示 2、现有工作单位收入3000元以上

申请所需材料：（复印件）

1、身份证（21—55周岁）

2、工作证明（劳动合同或公司开据工作收入证明需加盖公司章）

3、收入证明（柜台打印的最近半年个人银行流水帐需加盖银行业务章）

4、住址证明（最近60天内任意水、电费单据）

※ 5、若有可提供其它财力证明(如：房产证，行驶证)

★ 业主贷款 ——专为有房产人士量身定做（注：2—15万）

申请条件：无

申请所需材料：（复印件）

1、身份证（21—55周岁）

2、房产证（红本房、按揭房、购房合同、等均可）

3、住址证明（最近60天内任意水、电费单据）

4、红本房需要提供最近6个月的银行流水

※ 5、若按揭房需提供最近3个月内银行还款记录

★ 企业贷款（或个体） ——企业小额贷款的最便捷的贷款渠道！（注：2—15万）

申请条件：1、中国人民银行有个人征信记录显示

2、企业营业注册时间1年以上

申请所需材料：（复印件）

1、身份证（21—55周岁）

2、企业证明（营业执照副本、场地租赁合同、公章）

3、收入证明（柜台打印的对公及个人银行账户最近6个月流水账需加盖银行业务章）

4、住址证明（最近60天内任意水、电费单据）

※ 5、若有可提供其它财力证明(如：房产证，行驶证)

证大速贷（薪金贷）1-15万

  1. 身份证、社保卡
  2. 工作收入证明（如劳动合同、工作证、工牌等）
  3. 最近4个月银行卡流水（打印日期10个自然日内）
  4. 近60日内有效居住证明（如水电费、电话费帐号; 煤气，有线电视、信用卡对帐单其一）

证大速贷（事业贷）1-50万 

  1. 身份证
  2. 工作收入证明
  3. 营业执照、特种行业相关证照
  4. 最近6个月个人及对公流水（注：打印日期10个自然日内）
  5. 近60日内有效居住证明（如水电、电话费帐号; 煤气，有线电视、信用卡对帐单其一）
  6. 经营场地租赁合同
  7. 最近2个月（最近2次）企业水、电费单及场地交租单据

证大速贷（房产贷）1-15万

  1. 身份证
  2. 工作收入证明
  3. 房产证（是按揭房或已抵押的要提供近3个月的还款流水和贷款合同或购房合同）
  4. 近60日内有效居住证明（如水电费；电话费帐号; 煤气；有线电视费；信用卡对账单等任一收据或发票，要打印显示该贷款人的详细住址地址和姓名，以上居住证明其一即可）

证大速贷（车主贷）1-15万 

  1. 身份证
  2. 工作收入证明
  3. 机动车登记证，行驶证，交通事故责任强制保险单（如：车辆已抵押可提供抵押贷款合同或购车贷款担保合同）
  4. 近60日内有效居住证明（如水电、电话费帐号; 煤气，有线电视、信用卡对帐单其一）
